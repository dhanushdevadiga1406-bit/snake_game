# Snake Game

A simple Snake game with a separate backend and frontend.

## Structure

- `backend/` - Node.js backend with a high score API
- `frontend/` - HTML, CSS, and JavaScript game frontend

## Run locally

1. Open a terminal in `backend/`
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the backend server:
   ```bash
   npm start
   ```

## Controls

- Arrow keys or `WASD` to move
- Click Restart to play again

## Notes

The backend stores high scores in memory. If the server restarts, scores reset.
open http://localhost:3000 in your browser
